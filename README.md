# Picnic CSS Boilerplate

A collection of ready-to-use HTML templates built with the lightweight [Picnic CSS](https://picnicss.com/) framework.

## Available Templates

- **Landing Page** (`index.html`) - A responsive landing page with hero section, features, call to action and more
- **Login Page** (`pages/login.html`) - Clean authentication page with all necessary fields
- **Dashboard** (`pages/dashboard.html`) - Admin dashboard layout with navigation and content areas

## How to Use

1. Clone this repository
2. Open any HTML file in your browser to preview
3. Modify the templates to fit your project needs

## Features

- Minimal and clean design based on Picnic CSS
- Responsive layouts that work across devices
- No additional dependencies required
- Easy to customize and extend

## Customization

The templates use Picnic CSS's default styling. To customize:

1. Add your own CSS in the `css/custom.css` file
2. Modify the HTML structure as needed
3. Add your own JavaScript in the `js/` directory
